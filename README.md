# Harish Velthuri — Final Professional Portfolio

Premium movie + commercial editor portfolio.

## Features
- Correct video frames for vertical reels and horizontal brand films.
- Sound-enabled featured player.
- Brand collaboration names added.
- Movie + commercial + professional editing content.
- Enhanced layout with transitions, reveal effects and cinematic styling.
- Vanilla JavaScript + Vite, no JSX parse errors.

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Notes
Use original exported MP4 files for even better quality in production. The included files are optimized for a downloadable ZIP.
